- Fixed missing textures appearing sometimes with terrain presets
- Fixed colors made with Colour And Modify Things from uchideshi34 not being copied
- Fixed a copy of a mix of assets and text not pasting the text at the correct place
- Fixed walls being impossible to delete after a copy + undo + redo
- Fixed menu closing when selecting a roof and changing its settings
- Fixed potential softlock when pressing echap to close the terrain popup
- Added pressing Echap to close the terrain popup on slots 9+ wihtout quitting the terrain tool
- Fixed a vanilla bug that reorders assets in a multiselection
- Fixed a vanilla bug that messes the selection if you alt tab while making a dragbox
- Improved FPS count with grid ruler (CTRL + R) activated
- Made the terrain circle brush and square brush sizes independant
- Embedded the 9+ terrain slots' data in the map file instead of creating a separate file in the user folder
- Compatibility with the mod version checker from _Lib

v1.3.0

New features:
- Added slots 9-16 and 17-24 in the Terrain Tool (compatible with Additional Search Options)
- Added a terrain preset option
- Added a copy/paste option by sets of 8 terrain slots
- Added Crop, Soft Crop, Edge Crop and Symmetry to the existing Free Transform modes (crop modes = only for objects)
- Added a max opacity slider for the Crop modes so your cropped part can be half transparent
- Added asset count and pack stats in the Map Gallery
- Added an option to purge unused packs in the Map Gallery
- Added an overlay option for objects instead of the default yellow hoverbox
- Added an overlay option for patterns instead of the default outline
- Added a way to find the most demanding sub-mods (press F10, wait a few seconds, press F10 again)
- Added a filter floatbar in the selection tool (togglable in the vanilla floatbar)
- Added Blending Modes for the Trace Image Tool
- Added a way to keep a line stick to cardinal points when Snap is disabled (press CTRL while drawing). Works with paths, walls, patterns and roofs. Doesn't work in Edit Points mode and Snap To Grid mode.

Changes and Improvements:
- Masked paths and walls vanilla line on hover in Overlay mode
- "Link Sliders" in the Compare Tool is now greyed out if there's only one reference layer
- Major FPS improvement, especially in the Selection Tool and the Terrain Tool
- Improved selection area and reduced selection conflicts in Free Transform mode
- Improved compatibility with Rotate and Jiggle by adding a "Rotate 90°" button on right click on objects in the selection tool, fixing the auto rotate conflict which rotated the asset while the right click menu was displayed
- Improved Eyedropper recognition when hovering paths and walls
- Improved Wall overlay and selection
- Improved the Eyedropper by also taking into account the layer of the asset

Fixes:
- Fixed a vanilla bug in which changing the layer of two objects on top of each other might cause the smallest one to be unselectable
- Fixed an issue with the dropdown layer menu closing automatically in the selection tool
- Fixed Shift + O (overlays) being activated in text boxes
- Fixed new Trace Image not being enabled by default on saved maps
- Fixed Compare Tool window not being displayed properly on 4K monitors
- Fixed selection box handles not working over other assets (especially paths)
- Fixed Paths Overlay not working correctly with modified paths via Color and Modify Things from uchideshi34
- Fixed Free Transform not working on vanilla shadows
- Fixed Free Transformed assets not rotating after a skew/distort/perspective transform
- Fixed Reset Free Transform resetting to default asset size and rotation even if you had changed it without using FT
- Fixed Eyedropper working over UI
- Fixed any shoftcut containing the F key enabling Favorites
